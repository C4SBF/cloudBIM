# RE1 BIM and RDF

The apartment is part of a highrise tower. The selected apartment with 11 spaces is used to experiment with the Semantic Model and a very small federatated BIM (Architectural, Mechanical, Electrical, Plumbing) to explore the relationships and properties of:

* location of assets in spaces (rooms)
* connection of assets to their systems
* electical connecions
* sensors connections
